# svelte boilerplate

Boilerplate for building Svelte web apps with tailwindcss.

### Getting started
```bash
npx degit betososa/svelte-boilerplate svelte-project
```

### Development
```bash
npm run dev
```

### Storybook
```bash
npm run storybook
```

### Test + watch
```bash
npm run test
```

### Show coverga
```bash
npm run cover
```
